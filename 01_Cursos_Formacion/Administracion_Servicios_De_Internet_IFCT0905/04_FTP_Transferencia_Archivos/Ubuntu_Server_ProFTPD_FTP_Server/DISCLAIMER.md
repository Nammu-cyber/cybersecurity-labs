# Disclaimer

Este laboratorio se ha realizado en un entorno local y controlado con fines formativos.

- No debe exponerse el servicio FTP directamente a Internet sin medidas adicionales de seguridad.
- Las contraseñas reales deben ocultarse en capturas y documentación pública.
- El uso de permisos `777` se conserva como parte del ejercicio original, pero en un entorno profesional se recomienda aplicar permisos más restrictivos y ajustados al caso de uso.
- FTP transmite información sin cifrar si no se configura TLS/FTPS. Para entornos reales, usar FTPS, SFTP o VPN.
